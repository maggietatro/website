WHAT IS JAVASCRIPT

JavaScript is a programming language to add functionality and complexity to web pages that aren't possible using just HTML and CSS. it is the considered the third essential component of web design. it can also be the most confusing to use for visual designers because it truly is computer programming.

here are few resources to get started with understanding JavaScript:

  • Mozilla Developers Network Introduction to JavaScript
  • Mozilla Developers Network Learning Pathway to JavaScript
  • w3schools JavaScript Tutorial

JAVASCRIPT TOOLS FOR DESIGNERS

as a visual designer, you will probably not be spending your time writing JavaScript code or programming the functional structures underneath your site. that's the work of a dedicated back-end developer who finds their enjoyment in code. there are, however, some really good off-the-rack tools that allow you to integrate a bit of JavaScript magic without having to delve into the code.

There are plenty of tools out there to be discovered. Here are a few I find interesting:

  • Anime.js - use it to build complex animations. here are some codepen experiments that use anime.js
  • AOS (Animate On Scroll) - to easily get cool scroll effects
  • Bounce.js - build complex animations
  • Chart.js - to create charts. lots and lots of cool charts
  • Drop.js - to create cool hover/drop-down effects
  • FitText.js - make your text scale responsively
  • Granim.js - interactive animated gradients
  • Iconate.js - animated icon transforms
  • Lettering.js - cool text effects
  • Magic Animations - lots of premade animation effects
  • Slick - amazing carousel tool with tons of options
  • Tippy - create cool tool-tips
  • Vivus - make it look like your SVG graphic is being drawn

CSS honorable mention. these are not JavaScript, but are still cool libraries to know about:

  • Animate.css - lots of cool prebuilt css animations
  • Hover.css - lots of cool prebuilt hover effects

JAVASCRIPT FRAMEWORKS FOR DEVELOPERS

frameworks are the larger libraries and systems that developers use to build fully functional sites and apps. they are built on JavaScript, but have their own particular terminology and workflows. as a front-end designer, you will probably not deal with these directly, but you will probably be working with back-end developers who use these systems. at the very least, it is worth becoming aware of these frameworks as a part of the web design industry.

A few popular systems include:

  • Angular - many Google services are built using Angular. McDonald's, UPS, and Apple Support as well
  • React - used to build Facebook, Instagram, Netflix, Dropbox
  • Vue - used to build big sites like Alibaba and Nintendo
