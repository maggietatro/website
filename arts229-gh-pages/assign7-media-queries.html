MEDIA QUERIES

experiment assignment due thursday of week six

You will create a single webpage that uses media queries to show three different responsive layouts based on screen size.

There's plenty of competing advice on what sizes to use for your breakpoints based on specific devices and systems. To keep things simple for us, use these screen sizes for your break points:

  • up to 600px (phone)
  • up to 1200px (tablet)
  • over 1200px (desktop)

I prefer the mobile-first approach to web design, so make sure everything works on a small screen first and then add in extras as the screen gets larger. Practically, this means you only need to specify style rules for two breakpoints (600 & 1200).

A few things I'm looking for:

  • A phone layout will be a singe column of content, while larger screens will allow for a mulitple column layout.
  • Your navigation can be a simple vertical list on a phone, but a horizontal navigation bar on a bigger screen.

Beyond the functional layout changes, have fun with how the site looks on the different sizes. You don't necessarily need to be visually consistent from one screen size to another. For example, the backgrounds and fonts could change wildly from one screen to another.

Have fun!!
