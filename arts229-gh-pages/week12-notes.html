STYLE GUIDES & PATTERN LIBRARIES

these are the rules and guides and notes on how a product, brand, site should be handled visually. these list colors and fonts, how to handle logos and other branding, etc., often with a rationale for why the rules exist. style guides can exist for both print and web. when it comes to web design, the style guide is often coded as a living example of how to implement the styles.

  • my collection of notes on style guides
  • creative bloq article on creating a website style guide
  • flywheel has a good list of elements to include in your style guide
  • google material design is a famous robust style guide system
  • mozilla has a neat style guide system
  • buffer has a well-organized style guide

MODULAR WEB DESIGN

the appeal of modular web design has been growing quite a bit lately. the fundamental idea is that you code small modules and elements that can be combined in any way as needed to build your site. this is in contrast to templates designs that are built to have content fitted into.

  • my collection of notes on modular design
  • design shack has a good primer on modular design
  • modular web design: designing with components
  • brad frost's atomic web design utilizes a good metaphor for this process

CSS FRAMEWORKS

a framework is a foundation system that has been built to help you create consistent websites. the idea is to have a set of modular elements that can be used to build your site. usually these systems are a set of styles for different HTML elements so you can have functional columns, layouts, buttons, typography, etc. right out of the box. each framework is unique and has its own internal logic that may or may not make sense to you.

  • my collection of notes on CSS frameworks
  • bootstrap is the most famous framework. it is bloated and may be more than you need for a simple site
  • skeleton is the framework i used to start the class website
  • bulma is a popular open source framework that uses flexbox

WEB DESIGN INSPIRATION

it can be scary to stare at a blank html page with no content and no sense of visual direction. so don't do that. take the time to do some visual research. there are plenty of places to get inspiration on how to approach web design.

remember, you are not looking to steal a site completely, but you may find some neat ways of integrating specific elements or a cool color scheme. many of the highlighted sites have lots of cool effects that you may not be able to create yet. that's fine. focus on simplicity and functionality in your work. as you gain experience, you will figure out how to do the wild stuff.

  • call to idea is a great place to get inspiration on specific components like headers, footers, nav, lists, etc.
  • awwwards is one of the oldest sites for web design inspiration
  • siteinspire is one of my favorite places to look for inspiration
  • web design inspiration is also very good
  • the best designs is presents the designs quite well. good tagging system to refine your exploration
  • css nectar has a site of the day showcase
  • webflow has a showcase of websites and apps for inspiration
  • admire the web is cluttered, but they do showcase nice sites
  • behance is always a good place to search for design inspiration
