EXTERNAL CSS

We have mostly focused on internal CSS where the styles are listed in the head of the HTML file. That works for single page designs, but if you have more than one page, it is best to use an external CSS file. The advantage is that you can change one file and it updates the style of all pages that reference the CSS file.

To specify an external CSS file, put this code into the head of all your HTML pages:

<link rel="stylesheet" href="path/filename.css">
