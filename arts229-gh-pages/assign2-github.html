NOTES

GitHub is a platform to store and share your code. It is a verstion-control platform, which means it saves every draft and change you make to your files. Branching allows you to make copies of files and alter the code to have different versions. Forking allows you to take someone else's code and make your own version. It is a powerful and popular system. This page explains in more detail.

Just this past summer, Microsoft bought GitHub as a way to go all-in on open-source software. This is a big deal. Here's another article about the $7.5 billion acquisition.
