FLEXBOX

experiment assignment due tuesday of week seven

You will create a single webpage that explores different flexbox arrangements. A demo html page template will be provided to get you started on the demo.

A few things I'm looking for:

  • A few different layout arrangements. You can add or remove divs in each section if you would like.
  • Different visual explorations in addition to the flexbox arrangments, however don't get too carried away with the visuals. Basically, build off of what you've learned so far in previous experiments.
  • A simple two column layout in the final section with content to make it look like a simple website.

If you are feeling brave, try creating different breakpoint variations to show how your layouts can change at different screen sizes.

Have fun!!
