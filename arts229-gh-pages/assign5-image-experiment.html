IMAGE EXPERIMENT

experiment assignment due tuesday of week five

As a review of basic HTML & CSS, we are going to be playing around with using images on a website. You can load images through HTML as actual objects that take up space, but you can also load images through CSS as backgrounds of other objects. We will explore both methods.

I will cover most of the concepts as a class demo. I expect that you will complete the demo with me and then explore on your own. There are plenty of resources in the weekly notes for you to dig deeper.

A few basic concepts we will be experimenting with:

  • loading repeating background images
  • setting an image to fit the size of its container
  • styling images using CSS
  • using color gradients as images

To challenge yourself:

  • use clipping tricks to add images to text or custom shapes
  • creating image hover effects

In the end, I would like for you to have a single html page that utilizes at least EIGHT different images/effects including both CSS backgrounds and HTML image elements. Content and organization is purely up to you.

NOTES

Since this is a design class, I will expect you to spend time developing the visual of your pages. We are going to focusing on the coding and technical portion of web design in class, however visual design is also important. You will be graded on both aspects for every assignment.
